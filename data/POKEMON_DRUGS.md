## Criteria for Inclusion

A drug "sounds like a Pokemon" if it meets one or more of these criteria:

1. **Syllable pattern** - Ends in -xa, -ro, -ta, -ix, -ra, -vy, -za, -ia, -la, -va
2. **Fantasy/Sci-fi vibe** - Could exist in a JRPG
3. **Battle cry potential** - Would sound epic if shouted
4. **Phonetic similarity** - Resembles an actual Pokemon name
5. **Legendary energy** - Sounds like it has 600+ base stat total
